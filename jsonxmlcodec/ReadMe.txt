После распаковки архива jsonxmlcodec замените source-директорию в файле .path.txt на желаемую и переместите файл в C:\
Скопируйте в домашнюю директорию /jsonxmlcodec/json нужные json файлы.
Запустите JSONtoXMLConverter.
Результат конвертации в папке /jsontoxmlcodec/xml- .xml-дубликаты файлов домашней директории  
Логи ошибок в разрезе файлов - в папке /jsontoxmlcodec/errors.

Системные требования:
Windows 10+
Python Kernel 3+
